# Movie-API-Project

A fully responsive web application designed to showcase a collection of movies with a sleek, modern user interface.

🚀 Features
Fully Responsive: Works perfectly on mobile, tablet, and desktop.
Modern UI: Clean design focused on user experience.
Live Search: (If applicable) Quickly find the movies you're looking for.
🛠️ Built With
HTML5 / CSS3 & JavaScript

Movies Stall is a dynamic, fully responsive web application designed for cinema enthusiasts to discover and explore films across various genres. Key Features: Categorized Browsing, Live Search, Dynamic Sorting and Immersive UI. A sleek, "movie stall" themed interface featuring high-quality poster art and interactive elements.
